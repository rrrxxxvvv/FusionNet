# FusionNet: Graph Neural Network for Feature Recognition

## Data Availability Statement

### Public Datasets Used in This Study

The following open-source datasets were used for training and evaluation of FusionNet:

#### Fusion 360 Gallery Dataset
- **Link**: https://github.com/AutodeskAILab/Fusion360GalleryDataset
- **Description**: Real-world CAD models from Fusion 360 Gallery

#### MFCAD Dataset
- **Link**: https://github.com/hducg/MFCAD
- **Description**: Synthetic CAD dataset for machining feature recognition

#### MFCAD++ Dataset
- **Link**: https://pure.qub.ac.uk/en/datasets/mfcad-dataset-dataset-for-paper-hierarchical-cadnet-learning-from/
- **Description**: Extended version of MFCAD with enhanced annotations

### Proprietary Dataset

#### IRFD (Industrial Real-world Feature Dataset)
- **Status**: Not publicly available due to industrial confidentiality
- **Description**: Self-built dataset containing real industrial parts with manufacturing features
- **Note**: This dataset contains proprietary industrial data and cannot be shared publicly
